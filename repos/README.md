# Repositories

Curated collection of GitHub repositories discovered during research.

## Structure

- **index.md** — Categorized catalog of all repositories
- **.template.md** — Template for new repo entries
- **{owner-repo}.md** — Individual repository summaries

## Adding Repositories

1. Open a [📦 Add Repository](https://github.com/kdcllc/AI-DrivenDevEnv/issues/new?template=content-repo.yml) issue
2. Paste the URL, pick a category
3. Copilot agent summarizes it and opens a PR
4. Review and merge
