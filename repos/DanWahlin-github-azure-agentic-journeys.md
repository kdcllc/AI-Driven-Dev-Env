## Summary

GitHub and Azure Agentic Journeys is a collection of guided, end-to-end journeys that take developers from an idea or spec document to a fully deployed application on Azure using AI coding agents. Each journey uses GitHub Copilot (CLI, editor, or cloud agent) to scaffold APIs, databases, and frontends, add AI capabilities via Microsoft Foundry, generate Bicep infrastructure, and deploy to Azure Container Apps or AKS. The repo includes five concrete journeys ranging from a full-stack AI marketplace app to deploying open-source tools like n8n, Grafana, and Apache Superset.

## Key Features

- Five ready-to-run agentic journeys covering full-stack apps, open-source deployments, and AI-enhanced workflows
- Supports any AI coding assistant (GitHub Copilot CLI, VS Code Copilot, Claude Code, Codex) via standard agents and skills
- Integrates Microsoft Foundry for AI search, recommendations, and chat assistant capabilities
- Bicep-based infrastructure generation and deployment targeting Azure Container Apps and AKS
- Multi-language support: Node.js, Python, .NET, Java, and Swift stacks all guided by the same agentic workflow

## Relevance to AI-Driven Development

This repo is a practical, hands-on reference for teams who want to adopt agentic AI throughout the entire development lifecycle — from scaffolding code to shipping infrastructure to Azure. It directly demonstrates how GitHub Copilot agents and skills reduce time-to-deploy while keeping developers focused on product logic rather than infrastructure wiring.
