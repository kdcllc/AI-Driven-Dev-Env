# GitHub MCP Server Registry

- **URL:** https://github.com/mcp
- **Source:** github.com
- **Date Added:** 2026-04-20
- **Category:** Other

## Summary

The GitHub MCP organization (github.com/mcp) is the official registry for Model Context Protocol (MCP) servers, curated and hosted by GitHub. It catalogs 95+ official and community-contributed MCP servers from providers such as Microsoft Azure, Notion, Stripe, Figma, Supabase, MongoDB, Sentry, Atlassian, and many others. Each server exposes a standardized MCP interface that lets AI assistants and coding agents connect to external services, databases, and developer tools through a single protocol.

## Key Takeaways

- `github.com/mcp` is the primary discovery hub for 95+ production-ready MCP servers spanning cloud infrastructure, databases, SaaS APIs, web automation, and developer tooling.
- Top-tier providers including Microsoft, Atlassian, Stripe, Figma, Notion, and MongoDB publish official MCP servers here, enabling AI agents to interact with these platforms natively.
- The registry acts as a single discovery point for extending AI assistants like GitHub Copilot with ready-to-use, standardized integrations.

## Relevance to AI-Driven Development

Direct access to the MCP server registry lets developers building agentic AI systems quickly discover and integrate pre-built connectors for the tools and services they already use. This significantly reduces the effort required to extend AI workflows with external context and actions.
