# Articles

Curated collection of technical articles discovered during research.

## Structure

- **index.md** — Categorized catalog of all articles
- **.template.md** — Template for new article entries
- **{article-slug}.md** — Individual article summaries

## Adding Articles

1. Open a [📰 Add Article](https://github.com/kdcllc/AI-DrivenDevEnv/issues/new?template=content-article.yml) issue
2. Paste the URL, pick a category
3. Copilot agent summarizes it and opens a PR
4. Review and merge
