# AI-Related Learning Resources

Curated links for building and understanding agentic AI systems. This toolkit is designed around these patterns and tools.

## Agent Frameworks & Design

- **[Building effective agents](https://www.anthropic.com/engineering/building-effective-agents)** – Anthropic's guide to designing agentic systems with LLMs; foundational reading.
- **[How to develop AI apps and agents in Azure](https://devblogs.microsoft.com/all-things-azure/how-to-develop-ai-apps-and-agents-in-azure-a-visual-guide/)** – Visual walkthrough of agent architectures on Azure.
- **[Microsoft Extensions AI](https://learn.microsoft.com/en-us/dotnet/ai/microsoft-extensions-ai#stateless-vs-stateful-clients)** – Stateless vs. stateful patterns for AI clients.
- **[Building AI Agents with Semantic Kernel](https://devblogs.microsoft.com/semantic-kernel/guest-blog-semanticlip-a-practical-guide-to-building-your-own-ai-agent-with-semantic-kernel/)** – Practical guide using Semantic Kernel.

## Multi-Agent Orchestration

- **[Squad](https://github.com/bradygaster/squad)** – Human-led AI agent teams; integrates with GitHub Copilot. See this toolkit for integration patterns.

## Retrieval-Augmented Generation (RAG)

- **[Vector Search and Document Chunking](https://learn.microsoft.com/en-us/azure/search/vector-search-how-to-chunk-documents)** – Chunking strategies for effective semantic search.
- **[Integrated Vectorization in Azure Search](https://learn.microsoft.com/en-us/azure/search/vector-search-integrated-vectorization)** – End-to-end RAG on Azure.
